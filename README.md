ProjectSnowden
==============

UWP RezNet File Sharing

Instructions
--------------

1. Clone the project and all its submodules

    git clone --recursive https://github.com/naveenOnarayanan/ProjectSnowden.git

2. Depending on the OS run specific scripts

    WINDOWS
    run-windows.bat

    OSX
    bash run-mac.sh

    LINUX
    bash run-linux.sh
